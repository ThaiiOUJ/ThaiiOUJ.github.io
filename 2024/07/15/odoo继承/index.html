track_visibility 参数选择：

**track_visibility='always'**：始终跟踪该字段的变化。

**track_visibility='onchange'**：仅在字段的值发生变化时跟踪。

**track_visibility='never'**：不跟踪该字段的变化。

返回按钮

在 model 下写一个名为 action_return 函数

标准的返回方法 ir.actions.act_window

  • string=”返回”: 按钮的显示文本为“返回”。
  • type=”action”: 这个按钮的类型是“action”，表示点击按钮后会触发一个动作。
  • name=”%(action_demo_odoo_tutorial)d”: 这是一个引用语法，用于查找并调用名为 action_demo_odoo_tutorial 的动作。% 符号表示这是一个外部 ID，d 表示动态查找。此动作通常会定义在某个模块中，执行后可能会返回到某个视图或记录。
  • class=”oe_link”: 该按钮应用了 Odoo 的链接样式。

打印报告按钮

  • name=”%(demo_odoo_tutorial.action_report_demo_tutorial)d”: 这个按钮的动作是调用名为 action_report_demo_tutorial 的打印报告动作。与前面的按钮类似，这里使用了外部 ID 来引用该动作。
  • string=”打印报告”: 按钮的显示文本为“打印报告”。
  • type=”action”: 也表示这是一个触发动作的按钮。
  • class=”oe_highlight”: 该按钮应用了 Odoo 的高亮样式，通常用于强调重要操作。
结论
  • 内嵌方法: 这两行代码使用了 Odoo 的动态动作调用机制，允许按钮触发特定的动作，比如返回某个视图或打印报告。
  • 动作定义: 这些按钮的行为和功能依赖于在后端定义的动作（如 Python 代码中定义的 ir.actions.report 或其他类型的动作）。
