1. ALTER TABLE 语法

2. ALTER TABLE 的操作选项

  • ALTER COLUMN: 修改列的定义，包括数据类型、是否允许 NULL 值、默认值等。

  • MODIFY COLUMN: 修改列的数据类型和属性，但不能更改列名。

  • CHANGE COLUMN: 修改列的名称和定义（包括数据类型、是否允许 NULL 值、默认值等）。

  • ADD COLUMN: 向表中添加新列。

  • ALTER TABLE student_info
    ADD COLUMN 家庭住址 varchar(100);  -- 添加名为家庭住址的新列

3. ALTER TABLE 的常用选项

  • FIRST: 将新列放置在表的第一个位置。
  • AFTER existing_column: 将新列放置在指定列后面。
  • CASCADE: 级联操作，修改或删除主表中的行，会同步修改或删除相关联的从表中的行。
  • RESTRICT: 限制操作，确保在执行 ALTER TABLE 时不会产生级联效应。
